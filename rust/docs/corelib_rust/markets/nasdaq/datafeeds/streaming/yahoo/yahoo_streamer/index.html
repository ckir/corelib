§Yahoo Streaming Module

Long-running Tokio task with panic-restart supervisor. Persists subscriptions + config in redb. Decodes PricingData protobuf. Silence detection with exponential backoff (60s → 3600s). WS ping every 30s. Logs in exact StrictLogger format.
